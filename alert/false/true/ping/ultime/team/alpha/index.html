Dossier : "Tous Fraudeurs ?"

Une analyse factuelle basée sur le rapport de la Team Alpha. Décryptage d'une controverse médiatique, où le "docu-fiction" brouille les pistes entre divertissement et réalité sociale.

Le Mythe : "Tout le monde triche"

Perception Médiatique

Les reportages de type "docu-fiction" créent une loupe déformante. En se focalisant sur des cas extrêmes et scénarisés, ils laissent penser que la fraude sociale est la norme parmi les bénéficiaires d'allocations.

45%
Perception Publique Estimée

Pourcentage de gens qui croient que la fraude est généralisée.

La Réalité : Une Infime Minorité

Statistiques Officielles (ONSS/ONEM)

Les chiffres officiels sont têtus. Les infractions graves ne concernent qu'une fraction marginale des dossiers. La grande majorité des erreurs sont administratives, non intentionnelles.

Données basées sur les taux d'infractions constatés lors de contrôles ciblés.

L'Échelle des Coûts : Le Vrai Poids Économique

Alors que les caméras se braquent sur la fraude sociale (souvent de survie), la fraude fiscale massive (entreprises, évasion fiscale) pèse infiniment plus lourd sur le budget de l'État. C'est le "deux poids, deux mesures" de l'indignation.
